## Install

### Maven

Maven dependency
```Maven
<dependency>
  <groupId>com.whilerain.tinychart</groupId>
  <artifactId>tinychart</artifactId>
  <version>1.0</version>
  <type>pom</type>
</dependency>
```
  
### Gradle

Gradle dependency
```Gradle
implementation 'com.whilerain.tinychart:tinychart:1.0'
```
